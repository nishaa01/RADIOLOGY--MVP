 ## HL7 - NOTES
 <br>
HL7 (Health Level Seven) is a set of international standards used for the exchange, integration, sharing, and retrieval of electronic health information.

##  Need of HL7
Hospitals use multiple software systems like:
- Hospital Information System (HIS)
- Radiology Information System (RIS)
- Laboratory Information System (LIS)
- Electronic Medical Records (EMR)

HL7 allows these systems to communicate with each other in a standardized format.

## HL7 Messages
HL7 messages are text-based and consist of segments separated by pipes (|).

Each segment starts with a three-letter code.

### Common HL7 Message Types are
- ADT: Admission, Discharge, Transfer
- ORU: Observation Result
- ORM: Order Message
